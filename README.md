# 📝 Lista de Tarefas (Kanban Simples)

Projeto de uma lista de tarefas inspirado no modelo Kanban (semelhante ao Trello), desenvolvido com **HTML, CSS e JavaScript puro**.

A aplicação permite organizar tarefas por prioridade, visualizando-as em colunas e interagindo de forma simples e intuitiva.

---

## 🎯 Proposta

O objetivo deste projeto é aplicar conceitos básicos de desenvolvimento web, incluindo:

* Manipulação do DOM com JavaScript
* Organização de dados em vetores
* Separação de responsabilidades (HTML, CSS e JS)
* Estilização dinâmica baseada em condições

---

## ⚙️ Funcionalidades

* ✏️ Criar novas tarefas
* 🔽 Definir prioridade:

  * Alta
  * Média
  * Baixa
  * Sem prioridade
* 📌 Exibição em formato Kanban (colunas)
* 🎨 Estilização visual por prioridade
* ✅ Marcar tarefa como concluída (clique)
* 🗂 Organização em vetores separados por prioridade

---

## 🎨 Layout

* Cor de fundo principal: `#1c3938`
* Tarefas sem prioridade: `#122F51`
* Cada tarefa é exibida como um **"post-it" interativo**
* Interface simples, limpa e intuitiva
